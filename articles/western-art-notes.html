Notes

The dataset was extracted from joining two sources, objects.csv and objects_term.csv, from the National Gallery of Art's open data. objects.csv contains one row per object (144,192 total) with standard metadata (title, classification, medium, creation date, etc), and objects_terms.csv contains curated tags assigned by NGA cataloguers (568,518 total, since each object can carry multiple tags) describing the theme, school, keywords, etc.

The two were joined on object ID, keeping only works that had both a theme tag and a school tag. Minor themes that collectively accounted for less than 10% of tagged works were dropped to keep the analysis focused. The final dataset contains 58,234 rows representing 51,842 unique objects (some works carry multiple theme tags) spanning the 1400s to 2000s. The primary losses from the original data were prints (of which only 40% had theme coverage) and Index of American Design objects (9% coverage overall). Specifically, the dataset focuses on paintings, drawings, sculptures, and photographs.

The dataset is still sound for analysis, since the objects that are lost are not random. Inherently, objects that were dropped were the ones the NGA's own cataloguers hadn't fully classified by theme. This likely means they're the works where the subject matter is ambiguous, not yet reviewed, or not applicable. The surviving ones are the ones the gallery itself considers well-described. Given the datasets provided by the NGA, it is impossible to remove all limitations created by each object in the gallery not having a theme/school tag. Consequently, some skewing and small inaccuracies cannot be solved given the research question and the nature of the NGA datasets.
